# Tetricide

## Project Info
- A Tetris-style simulation used as a teaching tool, not a game
- Purpose: show people how AI changes the nature of their work — the easy pieces get removed progressively, the pace speeds up, and the remaining work overwhelms people
- Single file: `index.html`
- Live site: https://tetricide.clarkeching.com (GitHub Pages, auto-deploys from GitHub)

## Workflow
- Don't push to GitHub until I say "deploy"
- Each push triggers a GitHub Pages deploy
- Pull before starting if working from a different machine

## Key Details
- Audience is non-developers in workshops/talks
- Frame as a simulation/demonstration, not a playable game
- Behaviour to explore over time: remove easy pieces, increase speed, let the board overwhelm
